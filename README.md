\# HEDLauncher



An unofficial, mod-first Minecraft launcher with built-in Fabric and Modrinth integration.



\## Authentication status

Microsoft account authentication is currently under development.

A temporary local profile mode is available to simplify development and testing.

## Installation

1. Download the latest release (.exe)
2. Run the installer
3. Choose an install folder
4. Launch and play

No Java installation required.
All dependencies are bundled inside the executable.
